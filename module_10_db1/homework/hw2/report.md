# 📊 Отчёт по анализу продаж телефонов по цветам

## 📌 Задача
Отдел маркетинга поставил задачу:
- Выяснить, какие цвета телефонов покупают чаще всего.
- Сравнить популярность красных и синих моделей.
- Определить самый непопулярный цвет.

Для анализа использовалась база данных `hw_2_database.db`, таблицы:
- `table_checkout` — содержит информацию о покупках (`id`, `date`, `phone_id`)
- `table_phones` — содержит информацию о телефонах (`id`, `name`, `description`, `colour`, `price`)

---

## 🔧 SQL-запросы

### 1. Частота покупок по цветам
```sql
SELECT p.colour, COUNT(c.id) AS total_sales
FROM table_checkout c
JOIN table_phones p ON c.phone_id = p.id
GROUP BY p.colour
ORDER BY total_sales DESC;

Самый популярный цвет — синий, с 500 продажами.

Красные телефоны покупают реже, чем синие: 429 против 500.

Самый непопулярный цвет — золотой, всего 28 продаж.
